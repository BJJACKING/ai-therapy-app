## 🚀 立即开始

### 1. 克隆仓库
```bash
git clone https://github.com/BJJACKING/ai-therapy-app.git
cd ai-therapy-app
```

### 2. 配置环境
```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，配置以下必需变量：
# - DATABASE_URL (数据库连接)
# - JWT_SECRET (JWT 密钥)
# - OPENAI_API_KEY (OpenAI API 密钥)
```

### 3. 启动数据库
```bash
# 使用 Docker（推荐）
docker-compose up -d postgres redis

# 或手动安装 PostgreSQL 和 Redis
```

### 4. 运行迁移
```bash
cd backend
npm install
npm run migrate
```

### 5. 启动服务
```bash
# 后端（终端 1）
cd backend
npm run dev

# 前端（终端 2）
cd frontend
npm install
npm start
```

### 6. 访问应用
- 前端: http://localhost:3000
- 后端 API: http://localhost:3000/api
